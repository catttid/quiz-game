# Quiz Game
这是一个有趣的问答小游戏，旨在通过简单的网页互动测试用户的知识水平。项目使用了 HTML、CSS 和 JavaScript 编写，练习前端开发基础和 DOM 操作。
## 🎯 功能介绍
- 欢迎界面（开始游戏）
- 问题显示（包含题目、选项、当前题号、分数）
- 答案选择与评分
- 进度条展示
- 结果页面（显示得分并可重新开始）
